# Weather Icon Style

- Canvas: SVG `viewBox="0 0 64 64"`.
- Stroke: `#2B2B2B`, 3px, rounded caps and joins.
- Palette: sun `#FFC83D`, moon `#FBE7B6`, day cloud `#BFD7EA`, night cloud `#7A8AA0`, rain `#4FA3E3`, snow `#FFFFFF`, lightning `#FFD23F`.
- Style: friendly cartoon shapes, thick outline, soft highlights, readable at small sizes.
- Runtime names must match `weatherCodeToIcon(...)`: `clear_day`, `clear_night`, `cloudy_day`, `cloudy_night`, `overcast`, `fog_day`, `fog_night`, `rain_light_day`, `rain_light_night`, `rain`, `snow_light_day`, `snow_light_night`, `snow`, `thunderstorm`, `sleet`.
